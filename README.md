# Hotel Performance Dashboard | Power BI Project

## Overview

This project analyzes the performance of an imaginary hotel chain using Power BI.

## KPIs

* Total Revenue
* Occupancy %
* ADR
* RevPAR
* Realisation %
* DSRN

## Features

* City / Room Type / Month / Week Filters
* Revenue Trends
* Occupancy by Category
* Property Level Performance

## Key Findings

* Luxury category generated higher revenue than Business category.
* Some weeks showed revenue decline.
* Occupancy remained relatively stable.
* Certain properties outperformed others.

## Tools Used

* Power BI
* Power Query
* DAX
* Ms Excel
* CSV Files

# Hotel Performance Dashboard | Power BI Project

## Overview

This project analyzes the performance of an imaginary hotel chain using Power BI.

## KPIs

* Total Revenue
* Occupancy %
* ADR
* RevPAR
* Realisation %
* DSRN

## Features

* City / Room Type / Month / Week Filters
* Revenue Trends
* Occupancy by Category
* Property Level Performance
